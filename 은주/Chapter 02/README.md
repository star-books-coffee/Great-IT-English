## 1. 초심자의 마음으로, HTML / CSS / JavaScript

- `building block` : 구성 요소
- `besides` : ~외에, 게다가
    - Apart from, Other than, Aside from
    - Other technologies besides HTML are …
- `aspect` : 속성
    - Links are a fundamental aspect of the Web
- `annotate` : 주석을 달다
- `set off` : 구분하다 (set apart), 분리하다
