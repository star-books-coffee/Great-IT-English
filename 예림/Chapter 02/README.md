## 3. 프론트엔드의 절대 강자, 리액트 (React)

### 리액트에 대해서

- 빈번한 데이터 변화로 안정적인 웹페이지 리렌더링이 필요한 현대 웹 프론트엔드에서, 업데이트가 필요한 DOM 요소만 효율적으로 업데이트하는 가상 DOM을 활용해 빠르고 안정적인 성능 자랑
- 컴포넌트화된 프론트엔드 요소 사용 → 코드 재사용성과 생산성 향상
- JSX를 사용해 자바스크립트 코드 내에서 HTML 레이아웃을 정의할 수 있음

| 단어 | 뜻 | 비고 |
| --- | --- | --- |
